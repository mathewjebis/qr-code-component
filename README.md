### What I learned

This was my first Frontend Mentor challenge. I learned how to properly center a card using flexbox on the body element instead of using fixed margins. I also learned to avoid using `<br>` tags and span hacks for spacing — letting text wrap naturally inside a fixed-width container is the cleaner approach.

### Continued development

I want to keep improving my CSS layout skills, especially flexbox and grid. Next I will try more Frontend Mentor challenges while continuing my React learning journey.
